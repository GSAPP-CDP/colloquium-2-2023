Prototypes

The first prototype began with a physical object or set of objects that can be placed throughout the space to prompt the user to engage with the AR versions of the object as an exercise in redefining the context of these objects by associating them with spaces and objects familiar to the user. (Which has rightly been critiqued as static.) In its current form it meets my desire for accessibility but fails to provide a sustained engagement with the history and reality of these cultural artifacts. In order for it to be more dynamic, this prototype will need to move outside of the home and into spaces where there is even greater tension once one of the Edo artifacts is placed there via augmented reality.

Moving Forward

  • Fill in the gaps regarding transfer of artifacts between entities to create a complete dataset that can be used for more detailed network analysis, detailed interactive refugee maps for all documented artifacts in the Digital Benin dataset
  • Spatialize the images
  • Create an accessible and dynamic platform for the data, that interacts with the users spatial reality. Possibilities include:
    • Pedestrian maps within the cities around the world that meld 3d and 2d
    • Older idea: overlaying modern and traditional homesettings in Edo over the world map where the functions of each area/room/country is based on the cultural artifacts present there
